# commands

## global

- /kit (v2)
- /ticket (v1 - gets kit for free, chooses to join or create a colony)
- /mars (lists colonies)
- /colony create OR apply #ID
- /colony #ID (colony details)
- /colonist #ID (colonist details)
- /info #ID (for everything)
- /status (public and private properties)
